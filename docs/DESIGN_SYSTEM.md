# Дизайн-система — VPN (общая для admin-web и windows-client)

Направление: **тёмная, строгая «инструментальная» эстетика** — приборная панель / терминал
наблюдения за сетью. Плотные данные, хайрлайны, моноширинные числа, сдержанная типографика.
НЕ мягкая/«ламповая»/скруглённая. Одна яркая «фишка» — маршрут до ноды; всё остальное тихое.

Цель — чтобы интерфейс читался как профессиональный сетевой инструмент (ближе к Bloomberg
terminal / Mullvad / аудио-приборам), а не как типовой SaaS-лендинг.

---

## 1. Палитра (hex — источник истины)

Фон/поверхности (холодный сине-чёрный, НЕ чистый чёрный):
- `--void`        `#0A0C10`  — базовый фон приложения
- `--slate`       `#12151C`  — панели/карточки
- `--graphite`    `#181C25`  — приподнятые поверхности, hover-строки
- `--hairline`    `#222835`  — тонкие границы (1px)
- `--edge`        `#2E3542`  — более заметные границы, разделители

Текст:
- `--frost`       `#E6EAF2`  — основной текст
- `--ash`         `#8A93A6`  — вторичный текст, подписи
- `--mute`        `#5A6273`  — приглушённый, плейсхолдеры, оси графиков

Бренд-акцент (один, используется скупо: primary-действия, линия маршрута, активная серия):
- `--ion`         `#3DA9FC`  — основной акцент (холодный синий)
- `--ion-dim`     `#1E5C8F`  — приглушённый акцент (заливки, неактив)

Статусы (ТОЛЬКО для состояния, не для декора), десатурированные:
- `--ok`          `#46C28A`  — connected / active / healthy
- `--warn`        `#E0A22E`  — degraded / near-limit
- `--alert`       `#E5564B`  — error / blocked / over-limit

Дата-виз (uPlot вторичные серии, если нужно >1):
- серия-1 = `--ion`; серия-2 = `#B98AFF` (приглушённый violet); серия-3 = `--ash`.
Держать максимум 2–3 серии на графике.

## 2. Типографика

Подключать через npm `@fontsource` (НЕ Google CDN — клиент в WebView2 должен работать
офлайн): `@fontsource/archivo`, `@fontsource/inter`, `@fontsource/jetbrains-mono`.

- **Display / заголовки / крупные числа:** `Archivo` (вес 600–700, отрицательный
  letter-spacing у крупных). Для hero-чисел допускается `Archivo` 700. Это «строгий
  инженерный» гротеск — НЕ Space Grotesk (он заезжен).
- **Body / UI:** `Inter` (400/500/600).
- **Mono / данные / пинг / ключи / таблицы чисел:** `JetBrains Mono` (400/500). Все
  числовые метрики (ms, GB, IP, кол-ва) — моноширинные, `font-variant-numeric: tabular-nums`.

Тип-шкала (rem): 0.6875 / 0.75 / 0.8125 / 0.875 / 1 / 1.25 / 1.625 / 2.25 / 3.25.
Eyebrow-лейблы: `JetBrains Mono`, 0.6875rem, `uppercase`, `letter-spacing: 0.12em`,
цвет `--mute`.

## 3. Форма, сетка, тон

- **Радиус:** строго малый. `--r-sm: 3px`, `--r-md: 5px`. Крупные блоки/таблицы — 0–3px.
  Никаких «пилюль» и больших скруглений (кроме круглой кнопки коннекта в клиенте).
- **Границы:** 1px `--hairline`; акцентные/активные — `--edge` или `--ion`. Углы резкие.
- **Тени:** почти нет. Глубина — через границы и фон (`--slate` → `--graphite`).
  Допустим один очень тонкий ambient-glow только под кнопкой connect в connected-состоянии.
- **Spacing scale (px):** 2, 4, 8, 12, 16, 24, 32, 48. Плотно, без воздуха «лендинга».
- **Сетка:** хайрлайн-дивайдеры, выраженные колонки, левый сайдбар (admin), компактная
  вертикаль (client). Метрики подаём как «приборные» ячейки: eyebrow-лейбл сверху,
  крупное mono-число, дельта/подпись снизу.
- **Иконки:** `lucide-react`, stroke 1.5px, размер 16–18px, цвет `--ash`/`--frost`.

## 4. Компоненты (shadcn-стиль: Radix primitives + Tailwind + cva)

Настроить Tailwind (v3, через postcss) + завести `components.json`, чтобы shadcn
CLI/MCP мог добавлять компоненты позже. Базовые компоненты держать в `src/components/ui/`.
Тема НЕ дефолтная shadcn — все токены из этого файла (через CSS-переменные в `:root`,
маппинг в `tailwind.config` `theme.extend.colors`).

Минимальный набор: `Button` (variants: primary=ion, ghost, danger, outline; размеры sm/md),
`Card` (slate, hairline border), `Input`, `Select` (Radix), `Table` (плотная, zebra через
`--graphite` на hover, числовые колонки mono+tabular), `Badge`/`StatusDot` (ok/warn/alert),
`Dialog` (для подтверждения удаления устройства/ноды), `Tabs`, `Tooltip`, `Skeleton`.

Состояния всегда видимые: focus-ring `--ion` 2px; disabled — снижение opacity; hover —
`--graphite`. Уважать `prefers-reduced-motion`.

## 5. uPlot — общий тёмный пресет

Один общий враппер `src/components/chart/UPlotChart.tsx` (useRef + useEffect, cleanup на
unmount, ResizeObserver для ширины). Общая фабрика опций `chartTheme.ts`:
- Фон прозрачный; grid/ticks: `--hairline`; оси/лейблы: `--mute`, `JetBrains Mono` 10px.
- Серия по умолчанию: stroke `--ion`, width 1.5; заливка — вертикальный градиент
  `rgba(61,169,252,0.18)` → `rgba(61,169,252,0)`. Точки выключены, кроме hover.
- Курсор: тонкая вертикальная линия `--edge`; tooltip — `--slate` с hairline, mono-числа.
- Без тяжёлой легенды; подписи серий — свои, снаружи графика.
- Пресеты: `areaSeries` (трафик во времени), `lineSeries` (пинг/латентность),
  `sparkline` (мини, без осей — для компактных ячеек клиента),
  `barSeries` (трафик по дням/устройствам — опц.).

Данные сейчас — **моки** (детерминированные генераторы в `src/mock/`), с пометкой `// TODO:
replace with API`. Точки подключения реальных данных описать комментарием рядом.

## 6. Сигнатурный элемент — «Маршрут» (RouteMap)

Главная визуальная фишка. SVG-компонент `RouteMap`: точка **YOU** → линия → точка **NODE**
(выбранная локация). По линии анимированно движется «пакет» (точка/штрих) при состоянии
connecting/connected. Линия — `--ion`; в disconnected состоянии линия пунктир `--edge`,
пакет не движется. Подписи концов — mono (город/страна, ping ms). Анимация через
`stroke-dashoffset` / requestAnimationFrame; выключается при `prefers-reduced-motion`.
- В **клиенте** — крупно, центр экрана connect.
- В **админке** (Nodes) — мульти-маршрут: ядро → несколько нод, толщина/цвет линии =
  нагрузка/латентность.

## 7. Экраны

### Admin-web (admin + cabinet, разделение по `is_admin`)
Каркас: левый **сайдбар** (иконка+лейбл, секция ADMIN видна только админам), верхний
бар (контекст страницы + пользователь/logout). Контент — плотная сетка приборных ячеек.

- **Login** — строгий двухколоночный: слева форма (email/пароль), справа тёмная панель с
  ambient RouteMap-мотивом и eyebrow-копирайтом. Без «маркетинга».
- **Dashboard (cabinet)** — ряд метрик: статус подписки, трафик использовано/лимит
  (area uPlot + бар), кол-во устройств / лимит, ping до ближайшей ноды (sparkline).
- **Subscription** — текущий план (большие mono-числа лимитов, дата окончания), карточки
  тарифов Basic/Standard/Pro (1/100GB · 3/300GB · 5/1TB) со строгим выделением активного.
- **Devices** — плотная таблица: name, platform, last seen, traffic (mono), статус
  (StatusDot). Действия: block/unblock, **удалить** (Dialog-подтверждение). Сверху —
  «X / LIMIT устройств» индикатор.
- **Admin · Users** — таблица пользователей: email, план, статус, кол-во устройств,
  трафик; поиск/фильтр; действия. (Backend admin-API ещё нет → мок-данные + `// TODO`.)
- **Admin · Nodes** — список нод (локация, статус, нагрузка, ping), uPlot throughput +
  latency по выбранной ноде, RouteMap ядро→ноды, add/remove (мок). 

### Windows-client (компакт 420×720, тёмный, WebView2)
Вертикальный, без сайдбара; верх — лого/локация, низ — состояние/действие.
- **Login** — компактная форма.
- **Connect (главный)** — крупный **RouteMap** (YOU→NODE), под ним круглая/строгая кнопка
  **Connect/Disconnect** с состоянием (disconnected/connecting/connected/error), индикатор
  статуса (StatusDot + ms). Ниже: селектор локации (Select), две приборные ячейки —
  **ping** (line sparkline) и **throughput/трафик сессии** (area sparkline), адрес SOCKS
  (`127.0.0.1:10808`) с кнопкой copy и подсказкой `curl --socks5 ...`. Logout — иконкой.

## 8. Правила реализации
- Тёмная тема — единственная (пока). Все цвета — из CSS-переменных, маппинг в Tailwind.
- Сохранить существующую логику: сторы (MobX), API-клиенты, контракт control-API клиента —
  НЕ ломать. Дизайн меняет презентацию + добавляет графики/виджеты.
- Доступность: focus-видимый, контраст AA для текста, `prefers-reduced-motion`.
- Никаких секретов; моки данных помечать `// TODO: replace with API`.
- Шрифты бандлить через `@fontsource` (офлайн), не CDN.
